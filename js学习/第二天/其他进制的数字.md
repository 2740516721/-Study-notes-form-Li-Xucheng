十六进制

在js中，十六进制的数字以 0x 开头。

例：

```
let a = 0x123;
console.log(a);//输出时是以十进制输出
```

八进制数字

如果需要表示八进制的数字，则需要以0开头。

例：

```
let a = 070;
```

二进制数字

如果需要表示二进制数字，则需要以0b开头。



注意：但不是所有浏览器都支持。



```
let a = "070";
a = parseInt(a);
console.log(a);
像以上这种字符串，有些浏览器会当成八进制来解析，有些会当成二进制来解析。
```

如何解决以上问题？

可以在parseInt()函数中传入第二个参数。

```
let a = "070";
//在parseInt()传递第二个参数，为指定返回值的进制。
a = parseInt(a,10);
console.log(a);
```
