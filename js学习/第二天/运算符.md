#### 一、算术运算符

|  +   |       加       |
| :--: | :------------: |
|  -   |       减       |
|  *   |       乘       |
|  /   |       除       |
|  %   | 取模（取余数） |
|  ++  |      递增      |
|  --  |      递减      |

##### 注意事项：

①遵循先乘除，后加减的原则。

②进行取模运算时，结果的正负取决于被模数（%左边的数），与模数无关（%右边）。

③尽量避免使用浮点数进行运算，因为可能会有偏差。若一定要使用浮点数运算，可以先把浮点数转换为整数进行运算，最后结果再转换为小数。

④使用“+”或“-”可以代表正数或者负数；例如：（+1）、（-1）.



##### 递增递减运算符相关知识：



使用方法：a++或a--或++a或--a。



##### 递增（递减）符号在变量前面的所代表的含义：



1、符号放在变量前面称前置递增（前置递减），它表示，返回计算后的结果，也就是返回变量自增加（自减少）后的值。

2、符号放在变量后面称为后置递增（后置递减），它表示，返回计算之前的结果，是没有发生自增（自减）后的值，也就是变量的原数值。

###### 例子：

```html
let a = 1; a ++ 

console.log(a);//a=2

 console.log(a++);//a=2

let b = 2; ++b

console.log(b);//b=3

console.log(++b);//b=4
```

#### 二、比较运算符

###### 比较运算符通常对两个数据进行比较，其结果是一个布尔值：true或者false。

|  >   |   大于   |
| :--: | :------: |
|  <   |   小于   |
|  >=  | 大于等于 |
|  <=  | 小于等于 |
|  ==  |   等于   |
|  !=  |  不等于  |
| ===  |   全等   |
| !==  |  不全等  |

##### ==、!=与===、!=== 的区别：

​	== 、!=  运算符会将两个相比较的数值自动转换为相同数据类型，再进行比较（如果相比较的两个数值数据类型不相同）；

​	===、!== 运算符不仅比较值，也比较数据类型是否相同，假如数据类型不同，则返回false。

#### 三、逻辑运算符

|  &&  |  与  | a和b的值都为true，则结果为true，否则为false          | a&&b   |
| :--: | :--: | ---------------------------------------------------- | ------ |
| \|\| |  或  | a和b的值都为false，则结果为false，否则结果为true     | a\|\|b |
|  !   |  非  | 若a为true，则结果为false；若a为false，则结果为true； | !a     |

###### 逻辑运算符是从左到右的顺序进行求值。

##### 逻辑运算短路情况：

1、使用 && 连接两个 表达式。

语法为：表达式1&&表达式2。

如果表达式1的值为true，则返回表达式2的的值；如果表达式1的值为false，则返回f表达式1的值。

2、使用 || 连接两个表达式。

语法为：表达式1 || 表达式2。

如果表达式1的值为true，则返回表达式1的值；如果表达式1的值为false，则返回表达式2的值。

##### 例：

```html
// 0 的布尔值
         if(0){
            console.log('true');
        }else{
            console.log('false');
        }     
        //0 的布尔值为false

        // 与 运算的短路现象，输出 0 
        console.log(0 && 123);

        // 或运算的短路现象，输出123
       console.log(123 || 222);
```

#### 四、位运算符

###### 位运算符用来对数据进行二进制运算，它会将比较符两边的数值转换成二进制，然后再进行运算，最后结果再转换成十进制数输出。

|  &   |  按位”与“  |  a&b  | 如果两个二进制位都为1，则结果为1，否则为0                    |
| :--: | :--------: | :---: | ------------------------------------------------------------ |
|  \|  |  按位”或“  | a\|b  | 如果两个二进制位有一个值为1，则结果为1，一个1都没有，则为0   |
|  ~   |  按位“非”  |  ~a   | 0的值取反1，1的值取反0                                       |
|  ^   | 按位“异或” |  a^b  | 如果两个二进制位的值都相同，则结果为1，否则为0               |
|  <<  |    左移    | a<<b  | a左移b位，右边空位补0，左边移走的空位舍去                    |
|  >>  |    右移    | a>>b  | a右移b位，左边空位根据原数的符号位补0或者1，原来是负数就补1，是正数就补0 |
| >>>  | 无符号右移 | a>>>b | a右移b位，丢弃被移出位，左边最高用0填补，不考虑原数正负      |

#### 五、赋值运算符

###### 赋值运算符是将赋值符号右边的值赋给左边。

|  =   | 赋值             |
| :--: | ---------------- |

| >>=  | 右移位并赋值     |
| >>>= | 无符号右移并赋值 |
|  &=  | 按位”与“并赋值   |
|  ^=  | 按位”异或“并赋值 |
|  !=  | 按位”或“并赋值   |

##### 例：

```
let a = 1;
a += 1;
//a +=1  => a= a+1; a=2;
```

#### 六、*元运算符

①一元运算符

###### 	一元运算符，只针对一个表达式进行操作。

​	例如：a++、a--、++a、--a

②二元运算符

​	二元运算符只针对两个操作数的运算符。

​	例如：a+b

③三元运算符

​	三元运算符是一种需要三个操作数的运算符。

##### 语法：

​	条件表达式？ 表达式1：表达式2

如果条件表达式1为true，则返回表达式1的结果。如果条件表达式为false，则返回表达式2的结果。

##### 例：

```
		let a = 5;
        a>4? console.log('a>4'):console.log('a<4'); 
        let sttr = a>0? '大于':'小于';
        console.log(sttr);
```
